# Secure Authentication System

This project implements a secure authentication system using Python Flask and JWT.

## Features
- User Registration
- Password Encryption using bcrypt
- JWT-based Login Authentication
- Protected Routes
- SQLite Database

## Technologies Used
- Python
- Flask
- Flask-JWT-Extended
- bcrypt
- SQLite

## How to Run
1. Install Python
2. Create virtual environment
3. Install dependencies
4. Run: python app.py

## Output
Secure login system with JWT authentication.
